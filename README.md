# Production-Ready RAG AI Agent

This project implements a production-ready Retrieval-Augmented Generation (RAG) AI agent using Python.
It enables intelligent document querying by combining semantic search with large language models.

## Features
- Document ingestion
- Vector-based semantic search
- LLM-powered response generation
- Modular and scalable architecture

## Tech Stack
- Python
- LangChain
- Vector Database
- LLM APIs
